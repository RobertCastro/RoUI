# F1-006: Introducir quality gates estaticos

- Estado: backlog
- Fase: 1
- Dependencias: F1-002, F1-003

## Objetivo

Incorporar validacion de CSS, formato, contratos de paquete y presupuestos de
tamano sin introducir herramientas que no tengan una responsabilidad clara.

## Criterios de aceptacion

- [ ] CSS fuente pasa reglas acordadas de Stylelint o validador equivalente.
- [ ] CI detecta artefactos generados desactualizados.
- [ ] Bundle y paquetes tienen presupuestos iniciales documentados.
- [ ] Fallos conocidos del baseline se convierten en tareas o gates.
- [ ] Los comandos locales y CI son los mismos.
