Iconos

Set Lucide (MIT) como sprite SVG: un solo archivo, sin librerías ni PNG, coloreable con currentColor y escalable.

Cómo usar

1 · Inyecta el sprite una vez (al inicio del <body> o vía fetch)

<!-- inline --> <svg style="display:none">…symbols…</svg> · o fetch('dist/icons.svg')

2 · Referencia el símbolo

<svg class="ro-icon"><use href="#ro-i-bell"></use></svg>

Tamaños: .ro-icon--xs 14 · --sm 16 · base 18 · --lg 20. El color se hereda del texto (currentColor).
