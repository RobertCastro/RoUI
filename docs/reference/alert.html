Alert

experimental

Mensaje contextual inline que informa sobre el estado de una tarea o sección.

Cuándo usarlo

  • Comunicar un estado persistente en el flujo de la página (éxito, error, aviso).
  • Dar contexto junto al contenido relacionado, sin interrumpir.

Cuándo no

  • Para notificaciones efímeras y flotantes: usa Toast.
  • Para un anuncio global de producto en la parte superior: usa Banner.

Anatomía

<div class="ro-alert ro-alert--info" role="status">
  <svg class="ro-icon ro-icon--sm ro-alert__icon" aria-hidden="true"><use href="#ro-i-info"></use></svg>
  <div class="ro-alert__body"><span class="ro-alert__title">Aviso.</span> Tu cohort empieza el 27 de abril.</div>
  <button class="ro-alert__close" aria-label="Cerrar aviso"><svg class="ro-icon ro-icon--xs" aria-hidden="true"><use href="#ro-i-x"></use></svg></button>
</div>
  • Usa role=status (cortés) para info/éxito y role=alert (asertivo) solo para errores urgentes.
  • El icono es decorativo: aria-hidden. El significado va en el texto.
  • El botón de cierre necesita nombre accesible (aria-label).

API

Clases

ClaseDescripción
.ro-alertContenedor base.
.ro-alert--infoVariante informativa.
.ro-alert--successVariante de éxito.
.ro-alert--warningVariante de advertencia.
.ro-alert--errorVariante de error.
.ro-alert--brandVariante de marca (primary).
.ro-alert__titleTítulo opcional en negrita.
.ro-alert__closeBotón de cierre.

Tokens que consume

--ro-radius-xl --ro-info --ro-success --ro-warning --ro-error

Sin JavaScript: es CSS sobre HTML nativo.

Teclado y accesibilidad

TeclaAcción
TabAlcanza el botón de cierre (si existe).
Enter / EspacioCierra el aviso.

Contrato de accesibilidad: toast

  • No dependas solo del color: usa icono y texto para el significado.
  • Reserva role=alert para errores que requieren atención inmediata.

Do / Don't

Sí

  • Empareja color, icono y texto para el significado.
  • Sé específico: di qué pasó y qué hacer.

No

  • No uses role=alert para mensajes no urgentes.
  • No comuniques el estado solo con color.

Madurez

experimental Apariencia o API sin garantías completas; puede cambiar sin aviso.
