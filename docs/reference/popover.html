Popover

experimental

Panel flotante no modal, anclado a su disparador, para detalle o acciones contextuales.

Cuándo usarlo

  • Mostrar detalle o controles secundarios junto a un elemento.
  • Contenido interactivo breve que no debe bloquear la página.

Cuándo no

  • Para una etiqueta de texto simple: usa Tooltip.
  • Para una tarea que exige atención total: usa Modal.

Anatomía

<div class="ro-dropdown" data-ro-disclosure-root>
  <button data-ro-disclosure-trigger aria-haspopup="dialog" aria-controls="p1" aria-expanded="false">Ver detalles</button>
  <div class="ro-popover" id="p1" data-ro-disclosure-panel role="dialog" aria-labelledby="p1-t" hidden>
    <div class="ro-popover__arrow"></div>
    <div class="ro-popover__title" id="p1-t">Detalle</div>
    <div class="ro-popover__body">…</div>
  </div>
</div>
  • El disparador declara aria-haspopup, aria-controls y aria-expanded.
  • El panel es role=dialog no modal; inicia con hidden.
  • Se cierra con Escape o clic fuera (disclosure-controller).

API

Clases

ClaseDescripción
.ro-dropdownContenedor posicionador (relative).
.ro-popoverPanel flotante.
.ro-popover--endAlinea el panel a la derecha.
.ro-popover__titleTítulo del panel.
.ro-popover__bodyCuerpo del panel.
.ro-popover__arrowFlecha decorativa hacia el disparador.

Atributos data

AtributoDescripción
data-ro-disclosure-rootContenedor disparador + panel.
data-ro-disclosure-triggerBotón que abre/cierra.
data-ro-disclosure-panelPanel controlado.

Tokens que consume

--ro-radius-xl --ro-shadow-md --ro-z-overlay --ro-white

Primitiva

import { createDisclosureController } from "@robertcastro/roui/primitives/disclosure-controller";

Teclado y accesibilidad

TeclaAcción
Enter / EspacioAbre y cierra el popover desde el disparador.
EscapeCierra el popover y devuelve el foco al disparador.
TabRecorre el contenido del panel mientras está abierto.

Contrato de accesibilidad: menu-popover-tooltip

  • No es modal: no atrapa el foco ni bloquea el fondo.
  • El disclosure-controller cierra al hacer clic fuera y con Escape.

Do / Don't

Sí

  • Ancla el popover a su disparador y ofrece cierre con Escape.
  • Manténlo breve; para tareas largas usa Modal.

No

  • No lo uses para información esencial permanente.
  • No lo trates como modal atrapando el foco.

Madurez

experimental Apariencia o API sin garantías completas; puede cambiar sin aviso.
