Acciones

Button

experimental

Dispara una acción de la interfaz. Se apoya en el elemento nativo <button>.

Cuándo usarlo

  • Ejecutar una acción en la página: enviar, guardar, abrir un diálogo.
  • Cuando el control no navega a otra URL.

Cuándo no

  • Para navegar a otra página o ancla: usa un enlace <a> con estilo de botón.
  • Para alternar un estado binario persistente: considera Switch o un botón con aria-pressed.

Anatomía

<button class="ro-btn ro-btn--primary" type="button">Guardar cambios</button>
  • El elemento base es <button> nativo: aporta rol, foco y activación por teclado.
  • Declara siempre type (button|submit|reset) para evitar envíos accidentales.
  • Un botón solo de icono necesita nombre accesible (aria-label) y el svg con aria-hidden.

API

Clases

ClaseDescripción
.ro-btnClase base obligatoria.
.ro-btn--primaryÉnfasis principal (color de marca).
.ro-btn--darkVariante sólida sobre fondo claro.
.ro-btn--softFondo tenue, énfasis medio.
.ro-btn--subtleSin fondo hasta el hover.
.ro-btn--ghostSolo borde; acción secundaria.
.ro-btn--smTamaño compacto.
.ro-btn--lgTamaño grande.
.ro-btn-iconBotón cuadrado de solo icono (requiere aria-label).

Tokens que consume

--ro-button-primary-bg --ro-button-primary-fg --ro-radius-lg --ro-focus-ring --ro-transition

Sin JavaScript: es CSS sobre HTML nativo.

Teclado y accesibilidad

TeclaAcción
EnterActiva el botón.
EspacioActiva el botón.
Tab / Shift+TabEntra y sale del control.

Contrato de accesibilidad: native-controls.md

  • No se necesita role: <button> ya expone role=button.
  • El estado deshabilitado usa el atributo disabled nativo.
  • El anillo de foco visible proviene de --ro-focus-ring.

Do / Don't

Sí

  • Usa un verbo claro y corto como etiqueta ("Guardar", "Enviar").
  • Reserva una sola acción primaria por vista.
  • Da nombre accesible a los botones de solo icono.

No

  • No uses un <div> con click en lugar de <button>.
  • No dependas solo del color para distinguir la acción primaria.
  • No deshabilites sin explicar por qué o cómo habilitar.
