Overlays

Drawer

experimental

Panel deslizante lateral o inferior (off-canvas) para navegación o contenido secundario, modal.

Cuándo usarlo

  • Menú de navegación en móvil (lateral).
  • Panel de detalle o filtros que se abre desde un borde (bottom-sheet).

Cuándo no

  • Para una confirmación puntual: usa Modal.
  • Para una pista breve: usa Popover o Tooltip.

Anatomía

<button data-ro-overlay-open="d1" aria-haspopup="dialog" aria-controls="d1" aria-expanded="false">Menú</button>
<div class="ro-drawer-root" id="d1" data-ro-overlay-root hidden>
  <div class="ro-drawer-overlay" data-ro-overlay-backdrop></div>
  <aside class="ro-drawer ro-drawer--left" role="dialog" aria-modal="true" aria-labelledby="d1-t" tabindex="-1">
    <div class="ro-drawer__header">
      <h2 class="ro-drawer__title" id="d1-t">Menú</h2>
      <button class="ro-btn-icon" data-ro-overlay-close aria-label="Cerrar"><svg class="ro-icon" aria-hidden="true"><use href="#ro-i-x"></use></svg></button>
    </div>
    <div class="ro-drawer__body">…</div>
  </aside>
</div>
  • Reutiliza el overlay-controller: mismo contrato que Modal (foco, Escape, restauración).
  • El backdrop (data-ro-overlay-backdrop) cierra al hacer clic fuera del panel.
  • El bottom-sheet añade ro-drawer__grip como asa visual.

API

Clases

ClaseDescripción
.ro-drawer-rootContenedor del overlay (inicia hidden).
.ro-drawer-overlayBackdrop de fondo.
.ro-drawerPanel deslizante.
.ro-drawer--leftVariante lateral izquierda.
.ro-drawer--rightVariante lateral derecha.
.ro-drawer--bottomVariante inferior (bottom-sheet).
.ro-drawer__gripAsa del bottom-sheet.

Atributos data

AtributoDescripción
data-ro-overlay-rootContenedor del overlay.
data-ro-overlay-openDisparador: id del drawer.
data-ro-overlay-closeControl de cierre.
data-ro-overlay-backdropFondo que cierra al hacer clic.

Tokens que consume

--ro-radius-2xl --ro-shadow-md --ro-z-overlay --ro-white

Primitiva

import { createOverlayController } from "@robertcastro/roui/primitives/overlay-controller";

Teclado y accesibilidad

TeclaAcción
Tab / Shift+TabCicla el foco dentro del panel (trap).
EscapeCierra el panel y restaura el foco.

Contrato de accesibilidad: dialog-drawer

  • role=dialog aria-modal=true, igual que Modal.
  • El backdrop cierra al hacer clic; el foco vuelve al disparador.

Do / Don't

Sí

  • Usa el lateral para navegación y el inferior para acciones en móvil.
  • Cierra con Escape, backdrop y botón.

No

  • No pongas flujos largos de varios pasos en un drawer.
  • No dejes foco alcanzable detrás del panel.

Madurez

experimental Apariencia o API sin garantías completas; puede cambiar sin aviso.
