 ├─ .ro-header__brand                     logo

 ├─ .ro-header__title                     título contextual (centro)

     ├─ .ro-btn-icon  (wrench)            herramientas

     ├─ .ro-dropdown  (.ro-avatar)       perfil ▸ .ro-menu--end

  • Contexto del problema
  • Requerimientos y restricciones
  • Criterio de aceptación
